# 🌌 3D 고정밀 태양계 시뮬레이터 (Solar System Simulator)

실제 천문 데이터(질량, 반경, 공전 속도, AU 등)와 **작용-반작용 4차 룽게-쿠타(RK4)** 물리 엔진을 기반으로 구현된 초거대 WebGL 우주 시뮬레이션입니다. 

</div>

## ✨ 주요 기능 (Key Features)

* **100% 리얼 스케일 동기화 (Real-time Physics & Unit)**
  * 현실의 1초가 시뮬레이션에서도 1초로 흐르는 리얼타임을 완벽 구현했습니다.
  * 거리 단위는 `AU(천문단위)`, 시간은 `Year(지구의 1년)`, 질량은 `Solar Mass(태양 질량 1배율)`의 물리 상수에 기초하여 중력 역학을 연산합니다.

* **초고속 기하급수 배율 (Logarithmic Speed Controller)**
  * 우주는 넓고 깁니다. 지루한 1배속을 넘어 최대 1초에 지구 시간으로 **10년 치 공전 연산**을 도약하는 수억 배 배속 슬라이더를 적용했습니다.

* **Zero-Allocation 연산 파이프라인 최적화 (Extreme Performance)**
  * 메모리 누수를 원천 차단하기 위해 연산 코어를 C 구조에 가까운 **`Float64Array`** 배열로 대체하였고, 행성 간 중력 척력을 $N(N-1)/2$로 단축하여 막대한 연산 비용을 획기적으로 줄였습니다. 

* **행성 추적 레이더 및 오토 줌 시스템 (Raycast Lock-on Target)**
  * 빛나는 행성 점(Spot)을 마우스 클릭(Raycaster) 한 번만 하면, 즉각적으로 해당 행성에 카메라가 편대비행 하듯 탑승하며 우주 공간을 활강합니다.
  * 허공을 클릭하면 자유 시점 카메라로 분리됩니다.
  * 시야 방해 최소화를 위해, 작은 행성 표면에 피사체가 다가가면 반짝이는 빛(Glow) 마커가 자동으로 사라져 리얼한 행성 관측을 유도합니다.

### 로컬 실행 
이 프로젝트는 난독화 빌드 배포판입니다. `index.html`을 브라우저를 통해 오픈하거나, 아래 커맨드로 구동할 수 있습니다.
```bash
python3 -m http.server 8080
```
브라우저에서 `http://localhost:8080`에 접속하여 화려한 태양계를 감상하세요.
